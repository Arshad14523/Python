
# Python Coding Practice — Day One

A small practice repository containing an introductory Python script.

## Files

- `Day one.Py` — a beginner script for Day One exercises.

## Prerequisites

- Python 3.8 or newer

## Run

Open a terminal (PowerShell or CMD) in this folder and run:

```powershell
python "Day one.Py"
```

## Notes

- File names are case-insensitive on Windows but keep the exact name shown above when running commands.
- Edit `Day one.Py` to try different exercises.
